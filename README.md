# SaMuf - Sanjana Unfiltered

A minimalist, personal lifestyle blog dedicated to exploring life's flavors, detours, and everything in between.

## Dashboard Features

- **Minimalist Dashboard:** A stationary, 100vh non-scrolling layout for a premium desktop experience.
- **Smart Responsive Design:** Seamlessly transitions to a scrollable layout on mobile devices.
- **Category Lists:** Streamlined, numbered post lists for Food, Travel, and Informal Experiences.
- **Custom Branding:** Styled as **SaM<span style="color: #c06b4e;">uf</span>** with a unique brand icon.

## How to Run Locally

You can view the blog using any static file server. For example, using Python:

```bash
python -m http.server 8080
```

Once running, navigate to `http://localhost:8080` in your web browser.
